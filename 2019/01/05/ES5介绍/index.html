ES5介绍

发表于 2019-01-05 | 更新于: 2019-01-05 | 分类于 JavaScript |
字数统计: 1.3k | 阅读时长 ≈ 5
ES5是09年发布 我们经常看到的语法一般都被转化为ES5的语法,因为目前浏览器还不能直接编译ES6语法

ECMAScript的介绍

  1. 它是由ECMA组织(前生为欧洲计算机制造协商会)指定和发展的脚本语言规范
  2. 而我们学的JavaScript是ECMA实现的,但术语ECMAScirpt和JavaScirpt平时表达同一个意思
  3. JS包含三个部分:

    ECMAScript(核心)
    BOM(浏览器对象模型)
    DOM(文档对象模型)

  4. ES的几个版本

    ES5 : 09年发布 我们经常使用的是ES5 但是现在主流的框架都是使用ES6的语法 所以ES6是我们前端必会的一个规范
    ES6(ES2015) : 15年发布,也称ECMA2015
    ES7(ES2016) : 16年发布,也称ECMA2016 (变化不大)

ES5介绍

  1. 理解

    除了正常运行模式(混杂模式),ES5添加了第二种运行模式:”严格模式”(strict mode)
    顾名思义,这种模式使得JavaScript在更严格的语法条件下运行

  2. 目的/作用

    消除JavaScript语法的一些不合理、不严谨之处,减少一些怪异行为
    消除代码运行的一些不安全之处,为代码的安全运行保驾护航
    为未来新版本的JavaScript做好铺垫

  3. 使用

    在全局或函数的第一条语句定义为:"use strict";
    如果浏览器不支持,只解析为一条简单的语句,没有任何副作用

  4. 语法和行为变化

    必须用var声明变量

禁止自定义的函数中的this指向windowd

创建eval作用域

对象不能有重名属性

JSON拓展

  1. JSON.stringfigy(obj/arr) 将对象/数组转化为JSON字符串
  2. JSON.parse(JSON字符串) 将JSON字符串转化为对象或数组

Object拓展

ES5给Object拓展了一些静态方法,常用的2个:

  1. Object.create(prototype,[descriptorts])

  • 作用:以指定对象为原型创建新的对象

  • 为新的对象指定新的属性,并对属性进行描述

  1. value : 指定值
  2. wirtable : 标识当前属性值是否是可修改的,默认是false
  3. configurable : 标识当前属性是否是可以删除的,默认是false

  4. enumerable : 标识当前属性是否能用for in枚举,默认是false

  5. Object.defineProperties(object,descriptorts)
    作用 : 为指定对象定义拓展多个属性

  6. get : 用来获取当前属性值的回调函数
  7. set : 监听当前属性值触发的回调函数,并且实参即为修改后的值
  8. 存储器属性 : setter,getter一个用来存值,一个用来取值

对象本身的方法:

  1. get propertyName(){} 用来得到当前值的回调函数
  2. set propertyName(){} 用来监听当前属性值变化的回调函数

一定要记住的是 使用get设置的拓展属性 只能使用set来监听拓展属性

Array拓展

ES5给数组新增的方法:

  1. Array.prototype.indexOf() 获取某个值在数组中的第一个索引
  2. Array.prototype.lastIndexOf() 获取某个值在数组中的最后一个索引
  3. Array.prototype.forEach() 遍历数组
  4. Array.prototype.map() 遍历数组返回一个新的数组 返回修改之后的数组
  5. Array.prototype.filter() 遍历过滤出一个新的子数组 返回条件为true

Funciton拓展

  1. Function.prototype.bind()

    作用:将函数内的this绑定为obj,并将函数返回

  2. 面试题:区别bind()、call()、apply()

    都能改变函数内this的指向
    call()/apply()是立即调用函数
    bind()是将函数返回
    call()/apply()的区别就是传参列表不同
